# nine_grid 目录说明

本目录是独立算法包，封装九宫格核心计算、模型定义和命令行入口。  
它是前后端共用的底层计算来源。  
这里的代码偏纯算法和数据结构，不承担 Web 交互，且 `calculator.py` 支持按公历日期或农历 tuple 直接计算魄数。

| 文件名 | 地位 | 功能 |
| --- | --- | --- |
| `__init__.py` | 包出口 | 暴露算法包公开符号 |
| `calculator.py` | 算法核心 | 计算九宫格结果与派生信息 |
| `cli.py` | 命令行入口 | 提供本地命令行调用能力 |
| `models.py` | 领域模型 | 定义算法输入输出结构 |
| `region_data.py` | 静态数据适配 | 提供地区数据读取与访问 |
